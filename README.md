# Warthunder IR Missile Simulation

![Sampled image](SimSample.png)

## Description

Proof of concept for re-creating Warthunder Game's Infra-red missile's behavior from laws of physics and data-mined values.

## Environment

- python3: 3.10.2
- matplotlib: 3.5.1
- numpy: 1.22.3

Run simulator.py
